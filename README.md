# Udemy-Kurs zu Git
um *kursiv* zu schreiben, packt man Worte in einfache *Sterne* oder _Unterstriche_
## Überschrift 2. Ordnung
Um **fett** zu schreiben, werden Worte in doppelte **Sterne** oder __Unterstriche__ gepackt.
### und der 3. Ordnung
Falls man ***fett und kursiv*** schreiben möchte, werden die Worte ___mit dreifachen Sternen oder Unterstrichen___ umschlossen.
